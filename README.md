# pipeline_scheduler_demo
A basic webapp which allows users to define rules for device scheduling which can be turned on and off. Please keep in mind that the base is coded with heavy use of AI, so please tell me if this is similar to an existing program!

**Instructions for use:**
1. Import the project from github
2. Open terminal and change directory to your local project directory
3. Run python app.py
4. Open up your web browser at http://127.0.0.1:5000
5. Enjoy!

To quit the application, use ctrl + C in the terminal.
